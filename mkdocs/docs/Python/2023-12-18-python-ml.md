## Practise

- list删除最后一个元素的区别

```python
a=[1,2,3]
c=a
a=a[:-1] #创建新的切片并且赋值给a，c的内容并没有改变

b=[1,2,3]
d=b
b.pop() #直接修改原来的内存值，d的内容也发生改变
```

- 排序

```python
# 对列表进行排序
arr=[[1,3], [4,1]]
arr.sort(key=lambda x: x[1]) #默认升序排序
arr.sort(key=lambda x: x[1], reverse=True) #按照降序排序
```

- 反向遍历list

```python
arr=list()
length = len(arr)

for i in range(1, length+1):
    print(arr[-i])
```

- 反转字符串

```python
s="abc"
s=[::-1]
```

- 使用heapp对dict进行排序

```python
import heapq
c_map = dict()

#对dict中的元素按照val排序，返回k大的keys
heapq.nlargest(k, c_map.keys(), key=lambda k:c_map[k])
```
